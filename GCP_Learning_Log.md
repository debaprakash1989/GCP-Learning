#### Types of Cloud Computing
	Public :: Renting an Appartment (No Ownership, No headache to Maintain anything, Shared Space)
		Who Uses -> Start up
	Private :: Buy an Own Villa (High level security & control, Come with a cost)
		Who Uses -> Banks
	Hybrid  :: Buy an Own Villa in a Shared Commnunity (If required you can use some shared space as well.)
		Who Uses -> Banks when store Customer Data in Private Cloud and Demographic details in Public cloud
	Multi-Cloud :: Buy multiple Appartment/Villa in shared Community. (Use Multiple Providers service)
		Who Uses -> Big enterprise so that they will not depend on on cloud provider. They want multiple Backup everytime.
