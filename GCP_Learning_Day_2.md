#### GCP - Cloud Solution from Google.
	Youtube, Gmail run in GCP. So same speed and efficiecy you will get when you will use GCP.
	Kubernatics :: Application runs on this
	Users :: Paypal, Snapchat, Spotify.
  Billing per second make it Cost effective.
#### Strong points for GCP :: AI, Bigdata & Kubernatics.
#### Services
	Compute :: VMs, Applications (Paas), Cloud Functions (serverless), Kubernetics Engine (Containers)
  	Storage :: Cloud Storage, Cloud SQL, Fire Store, BIG Query
  	Networking :: VPC, Load Balancers, CDN
  	AI/ML :: Vertex AI, Vision API, Natural Language API
#### GCP Global Infrastructures
  	Region :: Geographical Area
	Zone :: Individual Data centers. To maintain Fault Isolation, Google build multiple Zone under Region.
  	Edge Network :: Facilities that bring Network & Content closer. They are not fully datacenter.
  	Multi-Regional Setup :: Redundancy accross Zone & location for Higher Availability.
